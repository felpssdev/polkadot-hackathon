# PolkaPay Frontend

Next.js frontend with 8-bit themed UI.

## Documentation

Complete frontend documentation available at [docs/frontend/](../docs/frontend/)

## Quick Start

```bash
# Using Docker
docker-compose up -d

# Local development
cd frontend
pnpm install
pnpm run dev
```
